How to use:


                selecter {
                    @include bubble-top(arg1, arg2, arg3);
                }

selecter: Specify the selector to which you want to apply the style.

arg1: Specify "background-color".

arg2: Specify "color".

arg3: Specify "left". My recommendation is from '20%' to '80%'. Please put "%" at the end.
